<h2 id="english">English Version</h2>

### 🔹 Features
- 🔐 **Authentication**: Secure login and signup system using Firebase Auth.
- 📅 **Booking System**: Select preferred dates and times for sports fields.
- 🏟️ **Venue Browsing**: Browse available sports venues and view details.
- 🧾 **Reservation Management**: View upcoming bookings and booking history.
- 📰 **News & Updates**: Stay informed with the latest announcements.
- 👤 **User Profile**: Manage user settings and profile information.
- 🌐 **Web Support**: Fully deployed and accessible via web browsers.

### 🛠️ Technologies Used
- **Frontend**: Flutter & Dart
- **Backend & Database**: Firebase (Authentication & Cloud Firestore)
- **State Management & Routing**: GetX
- **Deployment**: Vercel (Web Version)

### 🚀 Deployment
This project is configured for seamless deployment on Vercel as a Flutter Web application. It utilizes a custom build script (`build.sh`) to download the Flutter SDK and pre-cache the Web SDK to optimize build times.

<h2 id="ภาษาไทย">ภาษาไทย</h2>

### 🔹 ฟีเจอร์หลัก
- 🔐 **ระบบสมาชิก**: สมัครสมาชิกและเข้าสู่ระบบอย่างปลอดภัยด้วย Firebase Auth
- 📅 **ระบบจองสนาม**: เลือกวันและเวลาที่ต้องการเข้าใช้บริการสนามกีฬาได้
- 🏟️ **ค้นหาสนาม**: แสดงรายการสนามกีฬาที่มีให้บริการพร้อมรายละเอียด
- 🧾 **จัดการการจอง**: ดูรายการจองที่กำลังจะมาถึง และประวัติการจองย้อนหลัง
- 📰 **ข่าวสาร**: ติดตามประกาศและข่าวสารอัปเดตใหม่ๆ
- 👤 **โปรไฟล์ผู้ใช้**: จัดการข้อมูลส่วนตัวและการตั้งค่าต่างๆ
- 🌐 **รองรับ Web**: สามารถเข้าใช้งานผ่านเว็บบราว์เซอร์ได้ 100%

### 🛠️ เทคโนโลยีที่ใช้
- **Frontend**: Flutter และ Dart
- **Backend & Database**: Firebase (ระบบล็อกอิน และฐานข้อมูล Cloud Firestore)
- **State Management & Routing**: GetX
- **Deployment**: Vercel (สำหรับเวอร์ชันเว็บ)

### 🚀 การ Deployment
โปรเจคนี้ถูกตั้งค่าให้รองรับการ Deploy ขึ้น Vercel ในรูปแบบ Flutter Web โดยใช้สคริปต์ `build.sh` ในการติดตั้ง Flutter SDK และทำการ Precache Web SDK เพื่อให้การ Build รวดเร็วและเสถียรที่สุด
